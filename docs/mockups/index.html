What it does (v1)

  • Create any kind of class — school, course, club
  • Mark daily attendance in a tap (auto‑saves)
  • Present · Absent · Late
  • Mark holidays — clear on the calendar
  • Roster with roll no. + guardian phone
  • Monthly Excel (.xlsx) & PDF export
  • Fully offline, all on‑device
  • Extensible export engine (CSV / new layouts drop in)

A few small calls left

  • Holiday scope — per‑class (pottery off but school on), or a shared “school holiday” for all? I lean per‑class.
  • Weekends — auto‑skipped from school‑days, or only days you explicitly mark holiday?
  • One person in two classes — separate entries per class (simplest), or a shared person? I lean separate.
  • App name — “YesMaam” stays, or just a working title?
